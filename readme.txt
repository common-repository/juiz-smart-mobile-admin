== Description ==

**NOTICE: WordPress 3.8 offers you a new administration area. This new interface is fully
responsive.
Juiz Smart Mobile Admin is now obsolete. Support for this plugin will no longer maintained**

Used with the default administration theme, this plugin offers you a smartphone support for your dashboard.
Some other plugins are already supported (part of WPML, Simple Image Sizes, Next Gen Gallery, etc.), but the diversity of the WordPress community can't permit the Juiz SMA plugin to be perfect.

Some features :

* Translation ready (EN, FR and ES already supported)
* Others translation : LT (Lithuanian - by Vincent G from <a href="http://www.Host1Free.com">Host1Free</a>)
* Can zoom in a specific area (iPhone and iPod tested).
* Jump links (from footer to content, from content to menu).
* Automatic slide in the content when you click in a menu item.
* Auto-hide Safari iPhone (iOS<5) URL bar.
* Improvement of tables display
* Admin bar support (option for shortcuts plugin)
* Other goodies...


Important : this plugin was only tested by myself in iPhone in a specific context. 
If you have bugs, please, contact the support (information inside the plugin). 
Other tests are made by generous users. (thanks)
This plugin should work with smartphone which understand CSS media-queries.
Here, i need your feedbacks !

If you say "it's broken", please tell me why ;)

== Installation ==

You can use one of both method :

**Installation via your Wordpress website**

1. Go to the **admin menu 'Plugins' -> 'Install'** and **search** for 'Juiz Smart Mobile'
1. **Click** 'install' and **activate it**
1. (optional) Configure the Plugin in **Appearance**

**Manual Installation**

1. **Download** the plugin (it's better :p)
1. **Unzip** `juiz-smart-mobile-admin` folder to the `/wp-content/plugins/` directory
1. **Activate the plugin** through the 'Plugins' menu in WordPress
1. It's finished !


== Frequently Asked Questions ==

= This plugin works in which interface ? =
This plugin uses CSS3 Media Queries, my scripts automatically detect the windows browser size and apply styles when the width is small enought.
It can works with a big monitor and a small windows browser size. (tested with a LG E2350V and Firefox 8 resized)


= I've got a personnal ou custom admin theme... =
Juiz SMA is made for the default theme... but maybe it'll work... try, and uninstall if it doesn't work.

= Nothing ? =
You can use the forum support to ask me something.
